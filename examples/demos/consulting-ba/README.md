# Consulting BA Demo

International-student style demo for a US consulting internship.

## Route

- Template route: `mbb-consulting`
- Page size: US Letter
- Document template: `consulting-executive`

## Run

```bash
npm run export -- --input examples/demos/consulting-ba/outputs/source-resume.md --out-dir examples/demos/consulting-ba/outputs --base-name targeted-resume --template-route mbb-consulting
```

## Outputs

- `outputs/targeted-resume.docx`
- `outputs/targeted-resume.pdf`
- `outputs/optimization-report.md`
- `outputs/qa-report.md`
