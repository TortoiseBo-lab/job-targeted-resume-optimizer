# UK Graduate Scheme Demo

International-student style demo for a UK business graduate scheme.

## Route

- Template route: `uk-graduate-scheme`
- Page size: A4
- Document template: `classic-ats`

## Run

```bash
npm run export -- --input examples/demos/uk-graduate-scheme/outputs/source-resume.md --out-dir examples/demos/uk-graduate-scheme/outputs --base-name targeted-resume --template-route uk-graduate-scheme
```

## Outputs

- `outputs/targeted-resume.docx`
- `outputs/targeted-resume.pdf`
- `outputs/optimization-report.md`
- `outputs/qa-report.md`
