# Google SWE Intern Demo

International-student style demo for a US Big Tech software engineering internship.

## Route

- Template route: `google-big-tech`
- Page size: US Letter
- Document template: `tech-clean`

## Run

```bash
npm run export -- --input examples/demos/google-swe-intern/outputs/source-resume.md --out-dir examples/demos/google-swe-intern/outputs --base-name targeted-resume --template-route google-big-tech
```

## Outputs

- `outputs/targeted-resume.docx`
- `outputs/targeted-resume.pdf`
- `outputs/optimization-report.md`
- `outputs/qa-report.md`
