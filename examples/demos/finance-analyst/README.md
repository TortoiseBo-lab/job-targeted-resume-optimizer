# Finance Analyst Demo

International-student style demo for a US finance internship.

## Route

- Template route: `investment-banking-analyst`
- Page size: US Letter
- Document template: `finance-compact`

## Run

```bash
npm run export -- --input examples/demos/finance-analyst/outputs/source-resume.md --out-dir examples/demos/finance-analyst/outputs --base-name targeted-resume --template-route investment-banking-analyst
```

## Outputs

- `outputs/targeted-resume.docx`
- `outputs/targeted-resume.pdf`
- `outputs/optimization-report.md`
- `outputs/qa-report.md`
