# Chinese Internet Product Demo

International-student style demo for a Hong Kong/China-educated candidate applying to a Chinese internet product operations internship.

## Route

- Template route: `chinese-product-ops`
- Page size: A4
- Document template: `chinese-professional`

## Run

```bash
npm run export -- --input examples/demos/chinese-internet-product/outputs/source-resume.md --out-dir examples/demos/chinese-internet-product/outputs --base-name targeted-resume --template-route chinese-product-ops
```

## Outputs

- `outputs/targeted-resume.docx`
- `outputs/targeted-resume.pdf`
- `outputs/optimization-report.md`
- `outputs/qa-report.md`
