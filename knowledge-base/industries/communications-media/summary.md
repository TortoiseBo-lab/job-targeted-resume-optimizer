# Communication, Media and Cultural Industries Resume

Abstracted from the newly added docs/sample-resume-knowledge CV sample folder. This node stores role, layout, and language rules only; it does not copy source resume text.

## Best for

- Target role: Communication / Media / Cultural Industries
- Target country: International
- Length: One page; portfolio link only when the user has real work samples.

## Section order

1. Contact
2. Education
3. Experience
4. Projects / Portfolio
5. Research
6. Leadership
7. Skills and Languages

## Writing formula

Action + communication output/channel + audience/stakeholder + editorial or research method + reach, quality, or decision value

## Language style

Clear, audience-aware, editorial, and commercially grounded.

## What to preserve

- Domain-specific nouns and evidence types.
- Clear fact boundaries when metrics or authority are missing.
- Conventional ATS-readable structure for export to PDF, Word, and copy.

## Avoid

- responsible for
- helped with
- worked on
- participated in
- various
- many
- good
- excellent
- hardworking
- detail-oriented
