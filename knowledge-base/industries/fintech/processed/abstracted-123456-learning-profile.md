# Abstracted learning profile: Financial Technology Resume

Abstracted from the newly added docs/sample-resume-knowledge CV sample folder. This node stores role, layout, and language rules only; it does not copy source resume text.

## Best for

- Target role: Financial Technology / FinTech
- Target country: International
- Length: One page with finance logic and technical implementation both visible.

## Section order

1. Contact
2. Education
3. Technical Skills
4. Finance / FinTech Projects
5. Experience
6. Leadership
7. Certifications

## Writing formula

Action + financial product/data/model + technical method + risk, efficiency, accuracy, or user/business value

## Language style

Technical, financially precise, product-aware, and evidence-driven.

## What to preserve

- Domain-specific nouns and evidence types.
- Clear fact boundaries when metrics or authority are missing.
- Conventional ATS-readable structure for export to PDF, Word, and copy.

## Avoid

- responsible for
- helped with
- worked on
- participated in
- various
- many
- good
- excellent
- hardworking
- detail-oriented
