# Academic CV

Academic CV rules convert role expectations into sections, keywords, and bullet formulas. Key language signals: publications, conference, teaching, grants, service, advisor, methodology, peer review.

## What this node is best for

- Source type: industry
- Target role: Academic CV
- Target country: general
- Resume length: Evidence-dependent academic CV length.

1. Contact
2. Education

4. Publications

9. Service
10. Skills

## Formatting rules

- fontRecommendation: Readable serif or sans-serif resume font; keep one family and two weights at most.
- margin: 0.5-0.75 inch for one-page resumes; 0.75-1 inch for academic CVs.
- bulletStyle: Single-level bullets, 1-2 lines each, parallel punctuation.
- dateFormat: Consistent month/year or year-only format, right-aligned when layout supports it.
- useColumns: false
- atsFriendly: true

## Writing rules

- Tone: Methodical, scholarly, and evidence-specific.
- Bullet formula: Role + institution/lab + scholarly method + output, publication, teaching, or service contribution
- Quantification: Prioritize publications, conference, teaching, grants, service evidence where supported by the user's experience. Attach a number, scale, frequency, audience, runtime, revenue, cost, accuracy, conversion, latency, or time-to-completion metric whenever the source material supports it. When exact metrics are absent, use bounded scope statements such as team size, dataset size, user group, deliverable count, or project duration. Never invent metrics. Mark missing metrics as an improvement question instead of fabricating proof.
- Avoid: generic industry buzzwords without proof, unsupported expertise claims, responsible for, helped with, worked on, participated in, various, many, good, great, hardworking, detail-oriented

## Reusable prompt snippet

- When applying Academic CV rules, preserve conventional section names, prioritize the most relevant evidence first, and do not add unsupported content.
- Rewrite bullets using Role + institution/lab + scholarly method + output, publication, teaching, or service contribution; each bullet should start with a strong verb, name the method or tool, include scope, and end with an outcome when evidence exists.
