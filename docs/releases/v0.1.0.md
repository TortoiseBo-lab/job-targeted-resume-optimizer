# v0.1.0 - Initial Public Release

This is the first public release of Job Targeted Resume Optimizer, an open knowledge package and Claude Code skill for target-company and target-role driven resume tailoring.

## What's Included

- Open resume optimization knowledge base for companies, industries, universities, and writing styles.
- 67 target-driven resume template routes.
- Machine-readable and human-readable targeted resume playbook.
- Claude Code skill packaging at `.claude/skills/job-targeted-resume-optimizer/SKILL.md`.
- Copy-paste Chinese prompt for no-code use with ChatGPT, Claude, Gemini, or other LLMs.
- Chinese quickstart guide for non-developers.
- Example integration notes for loading playbook and template routing context.
- Data card, security policy, contribution guide, issue templates, and PR template.
- Validation script for release checks.

## Intended Use

Use this package to help an AI assistant:

1. Read a candidate resume and target job description.
2. Extract target role signals.
3. Map candidate evidence to direct matches, transferable matches, and gaps.
4. Choose a resume structure and template route.
5. Rewrite resume bullets without inventing facts.
6. Produce a targeted resume draft and optimization report.

## Known Limitations

- This is a v0.1 release, not a full production resume builder.
- It does not include the production web app.
- It does not include private corpora, user resumes, payment code, API keys, or deployment assets.
- DOCX and PDF generation are described as a workflow but are not yet bundled as a stable exporter.
- Real end-to-end examples are still planned for a future release.
