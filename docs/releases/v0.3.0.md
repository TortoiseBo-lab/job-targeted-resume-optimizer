# v0.3.0 - Installable Codex And Claude Code Skills

This release packages the project as an installable skill for both Codex and Claude Code.

## What's New

- Added npm CLI installer:

  ```bash
  resume-skill install --target all
  resume-skill install --target codex
  resume-skill install --target claude
  ```

- Added local install scripts:

  ```bash
  npm run install:skills
  npm run install:codex
  npm run install:claude
  ```

- Added Codex skill package:

  ```txt
  skills/codex/job-targeted-resume-optimizer/SKILL.md
  ```

- Added Claude Code skill package:

  ```txt
  skills/claude/job-targeted-resume-optimizer/SKILL.md
  ```

- The installer bundles:
  - Skill instructions.
  - Knowledge base.
  - Word/PDF/HTML export script.
  - Example inputs and outputs.

## Intended UX

Users install the skill, then ask Codex or Claude Code for a target-role resume package. The assistant follows a product-like flow:

1. Intake resume and JD.
2. Analyze target company and role.
3. Map evidence.
4. Choose strategy and template route.
5. Generate Markdown resume.
6. Export Word and PDF.

## Safety

The package does not modify Codex or Claude Code during `npm install`. Users must explicitly run the installer command.
