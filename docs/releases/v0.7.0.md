# v0.7.0 - Template-Level Rules And Export QA

This release upgrades the template system from document families to per-template specifications.

## What's New

- Expanded the callable template catalog from 67 routes to 96 routes.
- Added `knowledge-base/templates/template-specs.json` with one independent spec per route.
- Each template spec now includes:
  - Page size: US Letter or A4.
  - Font rules, including Word fonts and PDF fallback fonts.
  - Margins, line height, density, and divider rules.
  - Section order and writing rules.
  - Banned AI-tone expression rules.
  - Export QA thresholds.
- Fixed regional page-size defaults:
  - US, US school, and US company templates default to US Letter.
  - China, UK, Europe, Australia, and Hong Kong templates default to A4.
- Export now reports template QA results for routed exports.
- Added `--strict-qa` to fail export when QA warnings are present.

## QA Checks

The exporter now checks:

- Page size matches the selected template.
- Font matches the selected template.
- Required core sections are present.
- Skills section exists.
- Bullet density is not too low.
- Page fill is not too sparse.
- Divider/visual rules are enabled.
- Common AI-tone phrases are absent.
