# v0.6.0 - Document Template Families

This release connects the 67 target-driven template routes to actual Word/PDF document styling.

## What's New

- Added `--template-route <id>` for catalog routes such as `google-big-tech`, `wharton-finance`, and `mbb-consulting`.
- Added `--document-template <family>` for direct style control.
- Added seven document style families: `classic-ats`, `tech-clean`, `academic-cv`, `finance-compact`, `consulting-executive`, `creative-modern`, and `chinese-professional`.
- Word output now changes font family by template:
  - Serif/formal routes use `Times New Roman`.
  - Tech routes use `Arial`.
  - Modern routes use `Aptos` with sans fallback.
  - Chinese routes include Chinese font fallback.
- PDF fallback now switches between standard PDF fonts such as `Times-Roman` and `Helvetica`.
- Divider line weight, color, capitalization, and minimum density are now controlled by document template.
- The bundled example now exports with `--template-route google-big-tech`.

## Examples

```bash
npm run export -- --input outputs/google-swe-intern-targeted-resume.md --template-route google-big-tech
npm run export -- --input outputs/finance-analyst-targeted-resume.md --template-route wharton-finance
npm run export -- --input outputs/consulting-ba-targeted-resume.md --template-route mbb-consulting
```
