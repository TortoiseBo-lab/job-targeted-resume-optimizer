# v0.8.0 - International Student Demo Gallery

This release adds open-source-safe demo cases that feel closer to international-student job applications.

## What's New

- Added five synthetic demo packs under `examples/demos/`:
  - `google-swe-intern`
  - `finance-analyst`
  - `consulting-ba`
  - `chinese-internet-product`
  - `uk-graduate-scheme`
- Each demo includes:
  - Original resume input.
  - Target JD input.
  - Final resume source.
  - Word output.
  - PDF output.
  - Optimization report.
  - QA report.
  - Demo README.
- Added `npm run export:demos` to regenerate all demo Word/PDF outputs.

All demo people and materials are synthetic.
